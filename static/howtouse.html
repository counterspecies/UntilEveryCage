How to Use This Map

This map exposes the infrastructure of animal exploitation across America. Here's how to navigate it and use it effectively.

Filter and Search

Use the controls below the map to focus your search:

Search Options

  • Search Bar: Enter facility names, company names, or animal types (e.g., "Tyson", "cattle", "chickens")
  • State Filter: Select a specific state or view all facilities nationwide
  • Reset Filters: Clear all filters and return to the default view

Facility Information

Click any marker to see detailed information:

  • Basic Info: Name, address, phone, establishment ID
  • Volume Indicators: Visual bars showing processing/slaughter volume
  • Animal Types: What animals are processed or tested
  • Coordinates: Exact latitude/longitude for precise location
  • Directions: Click "Get Directions" for Google Maps routing

Understanding the Data

Data Sources

  • USDA Locations: Slaughterhouses and processing plants
  • APHIS Data: Animal testing laboratories
  • Inspection Reports: APHIS Licensed breeders, dealers, and exhibitors

Legal and Ethical Use

  • All data comes from public government records
  • Use this information responsibly and legally
  • Perfect for research, education, and peaceful activism
  • Respect private property and local laws

Getting the Most Out of the Map

For Researchers

  • Use state filtering to analyze regional patterns
  • Search by animal type to study specific industries
  • Export coordinates for GIS analysis

For Activists

  • Find local facilities for organizing
  • Create targeted campaigns using volume data
  • Share specific facility information with supporters

For Educators

  • Show students the scale of animal exploitation
  • Use volume indicators to discuss industry consolidation
  • Compare facility density across different regions

Reporting Issues

If you find problems with the map or data:

  • Incorrect facility locations
  • Missing facilities you know exist
  • Technical bugs or performance issues
  • Data that seems outdated
